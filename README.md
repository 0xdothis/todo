## 🚀 Getting Started

1. Prerequisites

- [Node](https://nodejs.org/en) V18 or later
- [pnpm](https://pnpm.io/)

2. Clone the repository

```bash
git clone https://github.com/your-username/todo.git
cd todo
```

3. Install Dependencies

```bash
pnpm install
```

4. Run backend server

```bash
cd backend
pnpm start
```
